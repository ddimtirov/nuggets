036/**
037 * <p><span class="badge green">Entry Point</span> Utility class providing methods for dealing with
038 * <a href="https://kotlinlang.org/docs/reference/exceptions.html#checked-exceptions">
039 * checked exceptions</a> and <a href="https://dzone.com/articles/filtering-stack-trace-hell">
040 * huge stacktraces</a>. Typically, you would want to import statically the family of {@code rethrow(...)}
041 * methods, and use the others through explicit class reference.</p>
042 *
043 * <p>All methods in this class are designed to be threadsafe.</p>
044 *
045 * <h2>String conversion</h2>
046 * <p>Every once in a while we need to format an exception without actually printing it.
047 * That is where {@link Exceptions#toStackTraceString(Throwable)} comes useful.</p>
048 *
049 * <h2>About Checked Exceptions</h2>
050 * <p>The theory says that we should either handle a checked exception by performing recovery/compensating action,
051 * or let it propagate up the stack. The problem is that often there is no sensible way to handle the exception, and
052 * while propagating sounds nice, for checked exceptions it also requires the methods at the top of the stack to
053 * declaring every checked exception that the invoked method may call. In the end, more often than not, they end up
054 * throwing {@code Exception} and not bothering with the details.</p>
055 *
056 * <p>Another popular approach is to wrap the checked exception in an unchecked one and rethrow. While that saves
057 * us the need to add a new exception to the declaration of every method up the call chain, it adds another unnecessary
058 * level of "caused by" exception, pushing the real exception further down in the log output.</p>
059 *
060 * <p>With this class, we propose a third approach - take advantage from the fact that the checked/unchecked exception
061 * difference is enforced by the Javac compiler and not by the Java VM. This means that if we could throw a checked
062 * exception from a method that does not declare it, we can cast away the "checkedness". To that end we add a set of
063 * methods facilitating the following use cases:</p>
064 *
065 * <ul>
066 * <li><a href="#uc1">Use Case 1: Rethrowing without declaring a checked exception</a></li>
067 * <li><a href="#uc2">Use Case 2: Enriching exception message with local information</a></li>
068 * <li><a href="#uc3">Use Case 3: Using methods throwing checked exceptions from non-throwing lambdas</a></li>
069 * <li><a href="#uc4">Use Case 4: Transforming rethrown exceptions</a></li>
070 * <li><a href="#uc5">Use Case 5: Setting default transformation for rethrown exceptions</a></li>
071 * </ul>
072 *
073 * <h2 id="uc1">Use Case 1: Rethrowing without declaring a checked exception</h2>
074 * <p>In the most simple case we would just want to throw a checked exception as if it were unchecked.
075 * Here is example of using the {@link #rethrow(Throwable)} to achieve this:</p>
076 *
077 * <pre><code>
078 * FooBar quietlyCreateFooBar() {
079 *     try {
080 *         return new FooBar();
081 *     } catch (Exception e) {
082 *         Exceptions.rethrow(e);
083 *         return null; // compiler doesn't know this line is actually unreachable, so we need return statement
084 *     }
085 * }
086 * </code></pre>
087 *
088 * <p>As the {@code rethrow()} method returns unbound generic type, it can be used as return value, regardless
089 * of the type, transforming the snippet above to the slightly shorter:</p>
090 *
091 * <pre><code>
092 * FooBar quietlyCreateFooBar() {
093 *     try {
094 *         return new FooBar();
095 *     } catch (Exception e) {
096 *         return Exceptions.rethrow(e);
097 *     }
098 * }
099 * </code></pre>
100 *
101 *
102 * <h2 id="uc2">Use Case 2: Enriching exception message with local information</h2>
103 * <p>Often, when a lower application layer throws exception, there is some extra piece of information
104 * that could help in debugging the issue. The standard way of handling that is by wrapping the exception
105 * in another exception and rethrowing. We suggest a lighter alternative of using {@link #rethrow(Throwable, String)}
106 * to prepend a string to the text of the original exception and rethrow as unchecked.</p>
107 *
108 * <pre><code>
109 * FooBar quietlyCreateFooBar(String templateId) {
110 *     try {
111 *         return new FooBar(templates.get(templateId));
112 *     } catch (Exception e) {
113 *         String layerInfo = "TemplateId=" + templateId + ":";
114 *         return Exceptions.rethrow(e, layerInfo);
115 *     }
116 * }
117 * </code></pre>
118 *
119 * <p>Think a bit before you choose between message enrichment and wrapping. When in doubt, use enrichment.</p>
120 *
121 *
122 * <h2 id="uc3">Use Case 3: Using methods throwing checked exceptions from non-throwing lambdas</h2>
123 * <p>Many APIs designed for Java8 lambdas make use of the {@code java.util.function.*} classes.
124 * This creates a problem when the lambda implementation throws exception. Consider the following
125 * code trying to map files to their sizes.</p>
126 *
127 * <p>Consider the streams API - a method to find all files larger than specified size would typically look as follows</p>
128 *
129 * <pre><code>
130 * Collection&lt;File&gt; findLargerThan(Collection&lt;File&gt; files, long size) {
131 *     return files.stream()
132 *                 .filter(it -&gt; {
133 *                     try {
134 *                          return Files.size(f.toPath()) &gt; size;
135 *                     } catch (IOException e) {
136 *                          throw new RuntimeException(e);
137 *                     }
138 *                 })
139 *                 .collect(Collectors.toList());
140 * }
141 * </code></pre>
142 *
143 * <p>By using {@link #rethrow(Callable)} the same can be rewritten more compactly as:</p>
144 *
145 * <pre><code>
146 * Collection&lt;File&gt; findLargerThan(Collection&lt;File&gt; files, long size) {
147 *      return files.stream()
148 *                  .filter(it -&gt; rethrow(() -&gt; Files.size(it.toPath()) &gt; size))
149 *                  .collect(Collectors.toList());
150 * }
151 * </code></pre>
152 *
153 * <p id="uc3a">There is another overload of {@link #rethrow(ThrowingRunnable)} that will be automatically
154 * inferred if the supplied lambda does not throw exception. Consider the following example :</p>
155 *
156 * <pre><code>
157 * void processFile(File f) throws Exception {
158 *      do something exceptional...
159 * }
160 * File processSafely(File f) {
161 *      rethrow(() -&gt; processFile(f));
162 *      return f;
163 * }
164 * </code></pre>
165 *
166 * <p id="uc3b">Often, we want to return use or return a particular value if the lambda finished without exception.
167 * In the case above, we want to return {@code f} - this is accommodated by one final overload
168 * {@link #rethrow(ThrowingRunnable, Object)}, shaving one line and improving the readability from the previous example:</p>
169 *
170 * <pre><code>
171 * File processSafely(File f) {
172 *      return rethrow(() -&gt; processFile(f), f);
173 * }
174 * </code></pre>
175 *
176 * <h2 id="uc4">Use Case 4: Transforming rethrown exceptions</h2>
177 * <p>The {@link ExceptionTransformerBuilder} API allows us fine grained control over the exceptions we throw.
178 * Use {@link #rethrowTransformed(Throwable, boolean)} to setup an ad-hoc transformation for rethrown exception.</p>
179 *
180 * <pre><code>
181 * FooBar quietlyCreateFooBar() {
182 *     try {
183 *         return dependencyInjector.instantiate(FooBar.class);
184 *     } catch (Exception e) {
185 *         return rethrowTransformed(e, true) // true means we want to ignore any default transforms (rather than adding to them)
186 *                .replaceMessage(t -&gt; t.getMessage() + " while creating foobar") // add extra info
187 *                .unwrapThese(InvocationTargetException, ExecutionException)     // remove unnecessary levels of indirection
188 *                .done();
189 *     }
190 * }
191 * </code></pre>
192 *
193 * <h2 id="uc5">Use Case 5: Setting default transformation for rethrown exceptions</h2>
194 * <p>While {@code rethrowTransformed()} gives us a lot of flexibility, sometimes we just want to set and forget
195 * an exception cleaning transformation for the whole application. That is what {@link #transformOnRethrow()} is
196 * about. If you configure a transformer with this method it will be applied automatically on every exception
197 * rethrown on the current thread and its children. That is, unless the application was started with {@code <b>-D</b>nuggets.Exceptions.transform-on-rethrow=false}
198 * command line parameter.</p>
199 *
200 * <p>You would usually want to call this method in the very beginning from your main method (which makes sure it would
201 * apply for all application threads). The method can be called any time and the transformation applies for the current
202 * thread and any future child threads.</p>
203 *
204 * <pre><code>
205 * Exceptions.transformOnRethrow()
206 *           .replaceStackTrace(ExceptionTransformerBuilder.FILTER_GROOVY_SYNTHETIC)
207 *           .filterStackFramesForClass(Pattern.compile("^(java.lang.reflect|sun.reflect).*"))
208 *           .filterStackFramesForClass(Pattern.compile("^(org.codehaus.groovy|org.gradle).*"))
209 *           .unwrapThese(InvocationTargetException.class)
210 *           .unwrapWhen(ex -&gt; ex.getMessage() == null &amp;&amp; ex.getClass().equals(RuntimeException.class))
211 *           .done();
212 * </code></pre>
213 *
214 * <p id="uc5a">The library even has a shortcut for the above, that you can apply by calling {@link #setupTransformOnRethrowDefaultConfig()}</p>
215 */
216public class Exceptions {
217    private static final boolean TRANSFORM_ON_RETHROW = Boolean.parseBoolean(System.getProperty("nuggets.Exceptions.transform-on-rethrow", "true"));
218    private static final InheritableThreadLocal<Function<Throwable, Throwable>> TRANSFORMER = new InheritableThreadLocal<>();

222    /**
223     * <p>Configures a transformer that would be used for every rethrown exception,
224     * unless a System property {@code nuggets.Exceptions.transform-on-rethrow}
225     * is not set explicitly to {@code false}</p>
226     *
227     * <p>This method sets up reasonable defaults that may change form version to
228     * version of this library. If you need more control, use {@link #transformOnRethrow()}
229     * instead.</p>
230     *
231     * @see #transformOnRethrow()
232     * @see <a href="#uc5a">Use Case 5 for example usage</a>
233     */
234    public static void setupTransformOnRethrowDefaultConfig() {
235        transformOnRethrow()
236                .unwrapPresetCommonWrappers()
237                .filterPresetReflection()
238                .filterPresetGroovyInternals()
239                .filterPresetGroovyScripts()
240                .filterStackFramesForClass(compile("^org\\.gradle\\..*"))
241                .build();
242    }

244    /**
245     * <p>Configures a transformer that would be used for every rethrown exception,
246     * unless a System property {@code nuggets.Exceptions.transform-on-rethrow}
247     * is not set explicitly to {@code false}</p>
248     *
249     * <p>The transformer is thread-local and inherited from children threads.
250     * This means that you'd better set it early in the application main thread.
251     * It is acceptable to redefine the transformer multiple times (i.e. worker
252     * threads wanting to clean up the thread-pool stack-frames)</p>
253     *
254     * <p>Keep in mind that this transformer is not automatically applied to the
255     * argument of {@link #toStackTraceString(Throwable)}, so one can easily dump the full
256     * stack if needed.</p>
257     *
258     * <p>If you are not sure what transformations to specify, you may want to just
259     * call {@link #setupTransformOnRethrowDefaultConfig()} at application startup,
260     * which would configure reasonable defaults.</p>
261     *
262     * @return a builder to configure the cleanup. Call {@link ExceptionTransformerBuilder#done()} to install.
263     *
264     * @see #setupTransformOnRethrowDefaultConfig()
265     * @see <a href="#uc5">Use Case 5 for example usage</a>
266     */
267    public static @NotNull ExceptionTransformerBuilder transformOnRethrow() {
268        return new ExceptionTransformerBuilder(TRANSFORMER::set);
269    }

271    /**
272     * <p>Rethrows exception, but not before cleaning it up with ad-hoc specified transformation.
273     * The transformation will be applied <em>before</em> OR <em>instead-of</em> the
274     * the default transformation, depending on the value of the {@code suppressDefaultTransform}
275     * parameter.</p>
276     *
277     * @param t an exception to format and rethrow.
278     *
279     * @param suppressDefaultTransform if {@code false} the default transformation, specified
280     *                                 by {@link #transformOnRethrow()} will be applied AFTER the
281     *                                 transform specified by this method. Otherwise, ONLY the
282     *                                 this transform will be applied (the default will be suppressed).
283     *
284     * @return a builder that would rethrow the formatted {@code t} when the
285     *        {@link ExceptionTransformerBuilder#done()} method is called.
286     *
287     * @see ExceptionTransformerBuilder
288     * @see #transformOnRethrow()
289     * @see #rethrow(Throwable, String)
290     * @see #rethrow(Throwable)
291     * @see <a href="#uc4">Use Case 4 for example usage</a>
292     */
293    public static @NotNull ExceptionTransformerBuilder rethrowTransformed(@NotNull Throwable t, boolean suppressDefaultTransform) {
294        return suppressDefaultTransform
295                ? new ExceptionTransformerBuilder(transform -> doSneakyThrow(transform.apply(t)))
296                : new ExceptionTransformerBuilder(transform -> rethrow(transform.apply(t)));
297    }

299    /**
300     * <p>Adapter allowing to use throwing methods from void lambdas of non-throwing interfaces.
301     * Unlike {@link #rethrow(ThrowingRunnable)} this overload returns a value and can be used
302     * in {@code return} statement. For usage examples, see {@link #rethrow(Callable)}.</p>
303     *
304     * @param throwingRunnable typically a parameterless lambda inside a capturing lambda of the desired type
305     * @param returnValueOnSuccess value to return if no exception is thrown.
306     *
307     * @param <T> inferring the return type.
308     * @return the {@code returnValueOnSuccess} if the {@code throwingRunnable} completed without throwing exception.
309     * @throws Throwable thrown from the {@code throwingRunnable} (undeclared)
310     *
311     * @see #rethrow(Callable)
312     * @see #rethrow(ThrowingRunnable)
313     * @see <a href="#uc3b">Use Case 3 for example usage</a>
314     */
315    @SuppressWarnings("JavaDoc") // declaring we throw an exception not in the signature
316    @Contract(pure=true)
317    public static <T> @Nullable T rethrow(@NotNull ThrowingRunnable throwingRunnable, @Nullable T returnValueOnSuccess) {
318        try {
319            throwingRunnable.run();
320            return returnValueOnSuccess;
321        } catch (Throwable e) {
322            return rethrow(e);
323        }
324    }

421    /**
422     * <p>Rethrows a checked exception as unchecked, even if it is not added to the method
423     * signature. Also allows to prepend some text to the exception message (useful when we
424     * want to capture some context without wrapping the exception).</p>
425     *
426     * <p>For more examples on the usage, see {@link #rethrow(Throwable) }</p>
427     *
428     * @param t an exception to rethrow as unchecked.
429     * @param prependedReason text to prepend to the exception's message
430     * @return dummy value, allowing this method to be used as a value of return statement.
431     * @param <T> return type - inferred by the compiler, depending on the context expectations.
432     * @throws Throwable thrown from the {@code throwingSupplier} (undeclared)
433     *
434     * @see Throwable#getMessage()
435     * @see #rethrow(Throwable)
436     * @see <a href="#uc2">Use Case 2 for example usage</a>
437     */
438    @Contract("_,_->fail") @SuppressWarnings("JavaDoc") // declaring we throw an exception not in the signature
439    public static <T> T rethrow(@NotNull Throwable t, @Nullable String prependedReason) {
440        if (prependedReason!=null && !prependedReason.trim().isEmpty()) {
441            Extractors.pokeField(t, Throwable.class, "detailMessage", prependedReason + t.getMessage());
442        }
443        return rethrow(t);
444    }

446    /**
447     * <p>Rethrows a checked exception as unchecked, even if it is not added to the method
448     * signature. Used when we don't want to recover from an exception and propagate it
449     * straight to the top-level exception handler.</p>
450     *
451     * <p>The return type is inferred, allowing us to use this method as return expression
452     * regardless of the return type and take advantage of the compiler reachability analysis.</p>
453     *
454     * @param t an exception to rethrow as unchecked.
455     * @return dummy value, allowing this method to be used as a value of return statement.
456     * @param <T> return type - inferred by the compiler, depending on the context expectations.
457     * @throws Throwable thrown from the {@code throwingSupplier} (undeclared)
458     *
459     * @see #rethrow(Throwable, String)
460     * @see <a href="#uc1">Use Case 1 for example usage</a>
461     */
462    @Contract("_->fail") @SuppressWarnings("JavaDoc") // declaring we throw an exception not in the signature
463    public static <T> T rethrow(@NotNull Throwable t) {
464        if (TRANSFORM_ON_RETHROW) {
465            Function<Throwable, Throwable> transformer = TRANSFORMER.get();
466            if (transformer!=null) {
467                t = transformer.apply(t);
468            }
469        }
470        //noinspection RedundantTypeArguments
471        return Exceptions.<RuntimeException, T>doSneakyThrow(t);
472    }

474    /**
475     * Uses a mix of type inference and generics to cast away the "checkedness" of exception.
476     *
477     * @param t an exception to rethrow as unchecked.
478     * @param <T> reinterpret-casting the desired throwing type.
479     * @param <R> inferring the return type.
480     * @throws T a type system loophole, using generic cast at the caller to make javac treat the exception
481     *           as {@code RuntimeException}.
482     *
483     * @see <a href="http://stackoverflow.com/questions/31316581/a-peculiar-feature-of-exception-type-inference-in-java-8">
484     *     Sneaky throw using Java 8 type inference</a>
485     */
486    @Contract("_->fail") @SuppressWarnings("unchecked")
487    private static <T extends Throwable, R> R doSneakyThrow(@NotNull Throwable t) throws T {
488        throw (T) t;
489    }

491    /**
492     * Conveniently formats the full stacktrace as a String.
493     *
494     * @param t the exception to format.
495     * @return a string representing the output of {@link Throwable#printStackTrace()}
496     */
497    public static @NotNull String toStackTraceString(@NotNull Throwable t) {
498        StringWriter sw = new StringWriter();
499        try(PrintWriter pw = new PrintWriter(sw)) {
500            t.printStackTrace(pw);
501        }
502        return sw.getBuffer().toString();
503    }

505    /**
506     * Parses a stack frame from its {@code toString()} representation.
507     * @param text containing the stack frame
508     * @param startIdx the number of bytes to skip (useful when processing exception lines)
509     * @return the reconstituted stack frame
510     */
511    public static @NotNull StackTraceElement parseStackFrame(@NotNull CharSequence text, int startIdx) {
512        int classEnd = -1;
513        int methodEnd = -1;
514        int fileEnd = -1;
515        int lineEnd = -1;
516
517scan_loop:
518        for (int i = startIdx; i < text.length(); i++) {
519            switch (text.charAt(i)) {
520                case '.':
521                    if (methodEnd<0) classEnd = i;
522                    break;
523                case '(':
524                    if (classEnd >= 0 && fileEnd<0) methodEnd = i;
525                    break;
526                case ':':
527                    if (methodEnd >= 0) fileEnd = i;
528                    break;
529                case ')':
530                    if (fileEnd >= 0) {
531                        lineEnd = i;
532                    } else {
533                        fileEnd = i;
534                    }
535                    break scan_loop;
536                default: break; // make findbugs happy
537            }
538        }
539
540
541        String declaringClass = classEnd<0  ? null : text.subSequence(startIdx   , classEnd ).toString();
542        String methodName     = methodEnd<0 ? null : text.subSequence(classEnd +1, methodEnd).toString();
543        String fileName       = fileEnd<0   ? null : text.subSequence(methodEnd+1, fileEnd  ).toString();
544        String lineNumberStr  = lineEnd<0   ? null : text.subSequence(fileEnd  +1, lineEnd  ).toString();
545
546        Objects.requireNonNull(declaringClass, "can't parse stack frame's declaringClass");
547        Objects.requireNonNull(methodName, "can't parse stack frame's methodName");
548
549        int lineNumber = lineNumberStr == null ? -1 : Integer.parseInt(lineNumberStr);
550        if (fileName!=null) {
551            switch (fileName) {
552                case "Native Method" : lineNumber=-2; break;
553                case "Unknown Source": fileName=null; break;
554                default: break; // make findbugs happy
555            }
556        }
557
558        return new StackTraceElement(declaringClass, methodName, fileName, lineNumber);
559    }

562     * <p>Reconstructs an exception graph (including type, message, causes, suppressed, stacktraces, hierarchy-loops)
563     * from a stacktrace. Does not populate any fields beypond what is in {@code Throwable}.</p>
564     * <p>If any exception is not available on the classpath, it is replaced with {@link MissingClassSurrogateException}
565     * that outputs the same {@code toString()} and {@code printStackTrace()} as in the parsed text.</p>
566     * <p>This method assumes that the stacktrace lines are delimited using the {@link System#lineSeparator()}</p>
567     *
568     * @param text the output of {@link Throwable#printStackTrace()}
569     * @return the throwable that produced the {@code text}
570     */
571    public static @NotNull Throwable parseStackTrace(@NotNull CharSequence text) {
572        return parseStackTrace(text, null);
573    }

575    /**
576     * <p>Reconstructs an exception graph (including type, message, causes, suppressed, stacktraces, hierarchy-loops)
577     * from a stacktrace. Does not populate any fields beypond what is in {@code Throwable}.</p>
578     * <p>If any exception is not available on the classpath, it is replaced with {@link MissingClassSurrogateException}
579     * that outputs the same {@code toString()} and {@code printStackTrace()} as in the parsed text.</p>
580     *
581     * @param text the output of {@link Throwable#printStackTrace()}
582     * @param eol line separator (i.e. when parsing files). If {@code null} will use {@link System#lineSeparator()}.
583     * @return the throwable that produced the {@code text}
584     */
585    public static @NotNull Throwable parseStackTrace(@NotNull CharSequence text, @Nullable String eol) {
586        String lineSeparator = eol==null ? System.lineSeparator() : eol;
587        String[] lines = text.toString().split(lineSeparator);
588        return rethrow(() -> parseStacktraceInternal(lineSeparator, "", new AtomicInteger(), new ArrayDeque<>(10), lines));
589    }

592     * Convert throwing runnable interface to normal runnable that rethrows silently.
593     * @param throwingRunnable runnable to be converted, may declare checked exceptions
594     * @return normal runnable that does not declare checked exceptions
595     */
596    @Contract("null->null;!null->!null")
597    public static @Nullable Runnable rethrowing(@Nullable ThrowingRunnable throwingRunnable) {
598        return throwingRunnable==null ? null : () -> rethrow(throwingRunnable);

611    private static final String STRACE_AT = "\tat ";
612    private static final String STRACE_SUPPRESSED = "\tSuppressed: ";
613    private static final String STRACE_CAUSED_BY = "Caused by: ";
614    private static final String STRACE_CIRCULAR_PREFIX = "\t[CIRCULAR REFERENCE:";
615    private static final String STRACE_CIRCULAR_SUFFIX = "]";
616    private static final String STRACE_SUPPRESSED_INDENT = "\t";
617    private static final Pattern STRACE_MORE = Pattern.compile(Pattern.quote(STRACE_SUPPRESSED_INDENT) + "*\t\\.\\.\\. (\\d+) more$");
618
619    private static @NotNull Throwable parseStacktraceInternal(
620            @NotNull String eol, @NotNull String prefix, @NotNull AtomicInteger index,
621            @NotNull Deque<Throwable> caused, @NotNull String[] lines
622    ) throws IllegalAccessException, InvocationTargetException, InstantiationException {
623        int i = index.get();
624
625        String[] classMessage = lines[i++].split(": ", 2);
626        String className = classMessage[0];
627        Class<Object> type = Extractors.getClassIfPresent(className);
628        Throwable t ;
629        try {
630            t = type == null
631                    ? new MissingClassSurrogateException(className)
632                    : (Throwable) type.getConstructor().newInstance();
633        } catch (NoSuchMethodException e) {
634            t = new MissingClassSurrogateException(className);
635        }
636        StringBuilder messageAccumulator = new StringBuilder(lines[0].length()); // one line messages are by far the most common case
637        if (classMessage.length > 1) messageAccumulator.append(classMessage[1]);
638
639        while (i < lines.length) {
640            if (!lines[i].startsWith(prefix)) break;
641            if (lines[i].startsWith(STRACE_AT, prefix.length())) break;
642            if (lines[i].startsWith(STRACE_CAUSED_BY, prefix.length())) break;
643            if (lines[i].startsWith(STRACE_CIRCULAR_PREFIX, prefix.length()) && lines[i].endsWith(STRACE_CIRCULAR_SUFFIX)) break;
644
645            if (STRACE_MORE.matcher(lines[i]).matches()) break;
646
647            messageAccumulator.append(eol).append(lines[i++]);
648        }
649
650        if (messageAccumulator.length()>0) {
651            String message = messageAccumulator.toString();
652            Extractors.pokeField(t, "detailMessage", message);
653        }
654
655        List<StackTraceElement> stackTraceAccumulator = new ArrayList<>(lines.length-i); // preallocate to avoid multiple resizing passes
656        while (i < lines.length) {
657            if (!lines[i].startsWith(STRACE_AT, prefix.length())) break;
658            StackTraceElement frame = parseStackFrame(lines[i++], prefix.length() + 4);
659            stackTraceAccumulator.add(frame);
660        }
661        if (i < lines.length) {
662            Matcher more = STRACE_MORE.matcher(lines[i]);
663            if (more.matches())  {
664                i++;
665                String duplicateFramesStr = more.group(1);
666                int duplicateFrames = Integer.parseInt(duplicateFramesStr);
667                Throwable lastCaused = caused.peekLast();
668                if (lastCaused!=null) {
669                    StackTraceElement[] causedStackTrace = lastCaused.getStackTrace();
670                    stackTraceAccumulator.addAll(Arrays.asList(Arrays.copyOfRange(
671                            causedStackTrace,
672                            causedStackTrace.length-duplicateFrames,
673                            causedStackTrace.length
674                    )));
675                }
676            }
677        }
678
679        StackTraceElement[] stackTrace = stackTraceAccumulator.toArray(new StackTraceElement[0]);
680        Extractors.pokeField(t, "stackTrace", stackTrace);
681
682        while (i<lines.length && lines[i].startsWith(prefix) && lines[i].startsWith(STRACE_SUPPRESSED, prefix.length())) {
683            String original = lines[i];
684            lines[i] = lines[i].substring(prefix.length() + STRACE_SUPPRESSED.length());
685            caused.addLast(t);
686            index.set(i);
687            try {
688                Throwable suppressed = parseStacktraceInternal(eol, prefix+ STRACE_SUPPRESSED_INDENT, index, caused, lines);
689                t.addSuppressed(suppressed);
690            } finally {
691                lines[i] = original;
692                i = index.get();
693                caused.remove(t);
694            }
695        }
696
697        if (i<lines.length && lines[i].startsWith(prefix)) {
698            Throwable cause;
699            if (lines[i].startsWith(STRACE_CIRCULAR_PREFIX, prefix.length()) && lines[i].endsWith(STRACE_CIRCULAR_SUFFIX)) {
700                String line = lines[i++];
701                String loopedToString = line.substring(prefix.length() + STRACE_CIRCULAR_PREFIX.length(), line.length()-STRACE_CIRCULAR_SUFFIX.length());
702                cause = caused.stream()
703                              .filter(it -> Objects.equals(loopedToString, it.toString()))
704                              .findFirst().orElseThrow(() -> new IllegalArgumentException("Can not resolve circular cause: " + loopedToString));
705
706            } else if (lines[i].startsWith(STRACE_CAUSED_BY, prefix.length())) {
707                caused.addLast(t);
708                index.set(i);
709                String original = lines[i];
710                lines[i] = lines[i].substring(prefix.length() + STRACE_CAUSED_BY.length());
711                cause = parseStacktraceInternal(eol, prefix, index, caused, lines);
712                lines[i] = original;
713                i = index.get();
714                caused.removeLast();
715            } else {
716                cause = null;
717            }
718            Extractors.pokeField(t, "cause", cause);
719        }
720
721        index.set(i);
722        return t;
723    }
